# 仿真部署方案

## 架构

```
模拟器（MuJoCo）渲染图像
        ↓
VLA 模型推理 → 7维末端位姿 Δ[dx,dy,dz,dr,dp,dy,grip]
        ↓
逆运动学（IK）将末端位姿 → 7个关节角度
        ↓
MuJoCo 执行关节角度
        ↓
渲染下一帧 → 循环
```

## 需要安装

```bash
pip install mujoco
# MuJoCo 会自动下载模型文件
```

## 实施内容

### 1. 加载 Franka 模型
DeepMind 的 mujoco_menagerie 提供 Franka Panda 的 MJCF 模型：
- 7-DoF 关节（panda_joint1 ~ panda_joint7）
- 末端执行器（panda_finger_joint1, panda_finger_joint2）
- 头部 RGB 相机

### 2. IK 求解器
MuJoCo 内置 `mujoco.mj_kinematics` + 逆运动学：
- 输入：当前关节角度 + 目标末端位姿
- 输出：目标关节角度
- 迭代求解，10ms 内收敛

### 3. 仿真循环

```python
for step in range(max_steps):
    # 1. 渲染 RGB 图像
    img = sim.render(camera="head", width=224, height=224)
    
    # 2. VLA 推理
    obs = [img, prev_img, prev_prev_img, prev_prev_prev_img]
    delta_pose = model.predict(obs, instruction)  # (7,)
    
    # 3. IK 求解目标关节角度
    current_qpos = sim.data.qpos[:7]
    target_pose = current_ee_pose + delta_pose[:6]
    target_qpos = ik_solve(sim, target_pose)
    
    # 4. MuJoCo 执行
    sim.data.ctrl[:7] = target_qpos
    sim.data.ctrl[7:9] = gripper_ctrl(delta_pose[6])
    for _ in range(substeps):
        mj_step(sim, model)
```

### 4. 无需训练
模型权重不变，IK 是纯几何求解，不需要任何训练。
